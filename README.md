# System-Root-for-Android-bushi-
System Root for Android(bushi)是一个基于HTML编写的模拟体验一键Root你的Android设备的娱乐网页。
我们使用了Material Design 3风格，优雅美观。
同时感谢仓库https://github.com/shiahonb777/web-to-app 提供的html转应用支持，使得我们可以为安卓端用户提供更为真实的体验。
